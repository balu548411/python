# Python Learning Repository

A comprehensive collection of Jupyter notebooks covering Python programming fundamentals and advanced topics.

## Overview

This repository contains educational Jupyter notebooks designed to teach Python programming from basics to advanced concepts, including data science libraries.

## Contents

### Fundamentals
- **00 Programming Environment** - Setting up the development environment
- **01 Hello World** - Introduction to Python
- **02 Variable Strings and Numbers** - Basic data types and variables
- **03 List and Tuples and Sets** - Collection data types
- **04 If Statements** - Conditional logic
- **05 While Loops and User input** - Loops and user interaction
- **06 Dictionaries** - Key-value data structures
- **07 Introduction to Functions** - Function basics
- **07 More Functions** - Advanced function concepts
- **08 Classes and OOP** - Object-oriented programming
- **09 Exceptions** - Error handling
- **10 External files** - File I/O operations
- **11 Decorators** - Python decorators
- **12 Persistence** - Data persistence techniques
- **Importing Modules** - Module system and imports
- **Coding Style (PEP8)** - Python style guidelines

### Data Science
- **13_NumPy** - NumPy library for numerical computing
- **14_Pandas** - Pandas library for data manipulation

## Requirements

- Python 3.x
- Jupyter Notebook or JupyterLab

## Usage

1. Clone this repository
2. Install Jupyter if not already installed:
   ```
   pip install jupyter
   ```
3. Launch Jupyter Notebook:
   ```
   jupyter notebook
   ```
4. Open the notebooks in sequential order for a structured learning path
